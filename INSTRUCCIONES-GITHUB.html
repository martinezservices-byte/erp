Publica tu ERP en Internet

Siguiendo estos pasos tendrás tu ERP en una URL propia, accesible desde cualquier celular, tablet o PC, gratis y para siempre. No necesitas saber programar — solo seguir las instrucciones.

1
Crear cuenta en GitHub
Si ya tienes cuenta, salta al Paso 2
ℹ
GitHub es el servicio de hosting más confiable del mundo — usado por Microsoft, Google y millones de empresas. GitHub Pages es completamente gratis y nunca expira.
1. Abre el navegador y ve a:
https://github.com
2. Haz clic en el botón verde Sign up
3. Ingresa tu email: alejandro@martinezservices.cl
4. Crea una contraseña segura
5. Elige un nombre de usuario — este será parte de tu URL:
Si tu usuario es:
martinezservices
Tu URL será:
martinezservices.github.io/erp
6. Completa verificación y confirma tu email
2
Crear un Repositorio
Es como una carpeta en la nube para tus archivos
1. Una vez dentro de GitHub, haz clic en el botón + New (esquina superior izquierda)
2. En "Repository name" escribe exactamente: erp
3. Selecciona ✓ Public (debe ser público para GitHub Pages gratis)
4. Marca la casilla ☑ Add a README file
5. Haz clic en Create repository
💡
El repositorio es la "carpeta" donde vivirán todos tus archivos HTML. Una vez creado, cualquier archivo que subas ahí será parte de tu sitio web.

Esta guía (opcional)
Para subir los archivos:
1. Dentro de tu repositorio recién creado, haz clic en Add file → Upload files
2. Arrastra y suelta TODOS los archivos de golpe (selecciónalos todos con Ctrl+A)
3. Espera a que todos se carguen (puede tomar 30-60 segundos)
4. En la parte inferior escribe un mensaje: ERP Martinez Services - versión inicial
5. Haz clic en Commit changes
⚠
Importante: El archivo que se llama index.html es el que GitHub mostrará automáticamente como portada del sitio. No le cambies el nombre.
4
Activar GitHub Pages
Esto publica tus archivos como sitio web
1. En tu repositorio, haz clic en la pestaña ⚙ Settings (parte superior derecha)
2. En el menú izquierdo busca y haz clic en Pages
3. En "Source" selecciona Deploy from a branch
4. En "Branch" selecciona main y luego / (root)
5. Haz clic en Save
6. Espera 2-3 minutos
7. Recarga la página Settings → Pages
8. Verás el mensaje: ✓ Your site is live at https://tunombre.github.io/erp

🔄
Cómo actualizar tu ERP
Cuando yo te haga mejoras o agregue módulos
1. Descarga el archivo nuevo o mejorado que yo te entregue
2. Ve a tu repositorio en github.com/tunombre/erp
3. Haz clic en Add file → Upload files
4. Sube el archivo nuevo (reemplaza automáticamente el anterior)
5. Haz clic en Commit changes
6. En 1-2 minutos el cambio está en vivo en tu URL ✓
✅
Los datos de tus empresas no se pierden — se guardan en el navegador de cada usuario (localStorage). Actualizar los archivos HTML no borra los datos.
